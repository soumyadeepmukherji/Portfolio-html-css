# Personal Portfolio Website

A modern portfolio website built using **HTML5** and **CSS3**.

This project is designed to showcase personal information, skills, projects, and contact details while maintaining clean structure and professional UI practices.

---

## 🚀 Features

- Semantic HTML Structure
- Modern CSS Styling
- Sticky Navigation Bar
- Hero Section
- About Me Section
- Skills Section
- Projects Showcase
- Contact Section
- Smooth Scrolling
- Clean and Organized Folder Structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 🎯 Purpose

This portfolio was created to practice and demonstrate:

- Semantic HTML
- CSS Layout Techniques
- UI Structuring
- Frontend Development Best Practices

---

## ⚙️ Installation

1. Clone repository

```bash
git clone <https://github.com/soumyadeepmukherji/Portfolio-html-css>
```

2. Open project folder

```bash
cd portfolio-html-css
```

3. Run locally

Open `index.html` in browser.
